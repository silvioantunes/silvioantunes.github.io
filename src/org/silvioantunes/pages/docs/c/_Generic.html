_Generic


A palavra-chave, _Generic, no C é uma extensão capaz de identificar os tipos dados.


O processo de identificar o tipo da variável acontece durante a compilação do código, o compilador identifica o tipo de dado, e realiza a substituição no código resultante. Permitindo a normal execução do programa sem incrementar tempo de execução.


Ela é excelente para criar bibliotecas, objestos, funções ou blocos de códigos que recebem vários tipos de dados diferentes, permitindo reutilizar códigos de maneira mais limpa e segura.
